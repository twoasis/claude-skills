# claude-skills

Shared Claude Code skill marketplace.

## Install (per teammate, once)

In **Claude Code**:

```
/plugin marketplace add twoasis/claude-skills
/plugin install html-to-elementor
```

Then the skill is available globally in every project — invoke with `/html-to-elementor`.

## Skills

- **html-to-elementor** — Reproduce an existing HTML webpage as pixel-exact, editable
  Elementor custom widgets in one WordPress plugin (fonts, spacing, colors, and all
  animations/interactions intact). Point it at a folder containing `final.html` + assets.

## Updating

Maintainer pushes changes here; teammates run `/plugin marketplace update claude-skills`
to get the latest.
